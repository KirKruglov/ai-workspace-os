## Проблема

Вы используете AI каждый день. Но каждая сессия начинается с нуля.

Агент не знает, кто вы, над каким проектом работаете, что решили на прошлой неделе
и в каком формате вам нужен результат.

Вы повторяете одно и то же. Получаете шаблонные ответы. Тратите время на правки вместо работы.

---

## Решение

AI Workspace OS — это система папок + набор шаблонов, которые дают AI-агенту
постоянную рабочую среду. Без кода, API или автоматизации.

Только текстовые файлы. Скопировал, заполнил, вставил. Всё.

```
global/
  about-me.md           ← кто вы, ваша роль, предпочтения
  global-rules.md       ← как агент должен себя вести и форматировать результат

projects/
  your-project/
    context/context.md  ← контекст проекта: цели, ограничения, решения
    logs/log.md         ← история сессий (агент пишет автоматически)
    input/              ← исходные материалы
    output/             ← финальные результаты

templates/
  session-start.md      ← вставьте в начале каждой AI-сессии
  task-brief.md         ← структурируйте задачу для лучшего результата
  agent-instructions.md ← мастер-prompt для любого AI-инструмента
```

---

## Как это работает

```
1. Заполните global/about-me.md       → расскажите AI, кто вы (один раз)
2. Заполните global/global-rules.md   → задайте стандарты работы (один раз)
3. Создайте папку проекта             → добавьте контекст под проект
4. Начните сессию с session-start.md  → агент читает всё, готов к работе
5. Ставьте задачи через task-brief.md → структурированный вход = структурированный выход
6. Агент логирует сессию              → история накапливается автоматически
```

Каждая новая сессия загружает ваш контекст. Без повторений. Без сброса.

## Быстрый старт

**1. Скачайте**

```bash
git clone https://github.com/KirKruglov/ai-workspace-os.git
```
Или: **Code → Download ZIP**

**2. Заполните профиль**

Откройте `global/about-me.md` и `global/global-rules.md` — заполните поля `[ ]`.

**3. Начните сессию**

Скопируйте `templates/session-start.md`, впишите название проекта и задачу на сегодня, вставьте в Claude или ChatGPT.

**→ Полное руководство:** [docs/quick-start.md](docs/quick-start.md)

## Почему это работает

**Постоянный контекст** — агент всегда знает, кто вы и над чем работаете.

**Структурированный результат** — единые правила = единое форматирование в каждой сессии.

**Память между сессиями** — агент логирует работу. В следующей сессии читает лог. Без амнезии.

**Не привязан к инструменту** — работает с любым AI, который принимает текстовый prompt.

**Без привязки к платформе** — обычные `.md`-файлы. Редактируйте в любом текстовом редакторе. Специальный софт не нужен.
